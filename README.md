# Convolutions and Pooling

This repository contains Python scripts covering image processing concepts using NumPy, specifically focused on convolution operations and pooling layers in deep learning.

## Requirements
- Executed on Ubuntu 16.04 LTS with Python 3.5
- Code style: `pycodestyle` (version 2.5)
- Libraries: `numpy` (version 1.15)

## Tasks
* **0-convolve_grayscale_valid.py**: Performs a valid convolution on grayscale images.
